# iTeda Solutions Platform

A Next.js-based platform for managing dryers and monitoring operations.

## Getting Started

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
npm start
```

### Database Setup
See `developer/DATABASE_SETUP.md` for detailed database configuration instructions.

## Documentation
Additional documentation for developers can be found in the `developer/` folder.

## Tech Stack
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Drizzle ORM
- PostgreSQL/Supabase
